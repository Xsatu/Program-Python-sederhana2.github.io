Keliling Persegi Panjang

Hai, saya akan membuat program python sederhana yaitu membuat program menghitung keliling persegi panjang. Caranya adalah sebagai berikut:
1. Buatlah folder dengan nama bebas.
2. Buka text editor kalian.
3.Masukan folder yang sudah kalian buat, dan buatlah file dengan nama bebas dan di akhiri .py seperti gambar dibawah ini.

Lalu kita membuat judul nya dengan mengetikan print("Program Python Menghitung Keliling Persegi Panjang")

Setelah itu kita buat input nya dengan mengetikan p = int(input("Panjang :")) lalu ketikan lagi l = int(input("Lebar :"))

Setelah membuat inputan sekarang kita buat rumusnya dengan mengetikan k = 2*(p+l)

Setelah sama seperti gambar di atas selanjut nya kita buat agar rumus dan inputan kita bisa berjalan dengan mengetikan print("Keliling Persegi Panjang = %0icm " %K)

Setelah selesai mengikuti langkah diatas mari kita coba dengan menggunakan cmd.
1.Buka cmd.
2. Ketikan cd Documents.
3.buka foldernya dengan mengetikan cd program python.
4. Masukan angka yang diinginkan
